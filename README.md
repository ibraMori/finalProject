GestionBug est une application WindowsForm Desktop de gestions de bug. 
Le projet respecte une architecture multi tier pour augmenter la maintenabilité du projet.
C# est utilisé pour le backend et FrontEnd et le projet utilise le .Net Framework.

Les Fonctionnalités:
  - Ajout d'un Bug avec une description
  - Afficher tous les bugs et trier avec des paramètres
  - Modification de Bugs
  - Supression de bugs
